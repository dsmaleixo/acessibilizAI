# components/

Componentes de UI reutilizáveis e sem regra de negócio.

- `ui/` — botões, inputs, campos de formulário acessíveis (rótulo + erro associado)
- `a11y/` — utilidades de acessibilidade (skip-link, live region, foco)
- `denuncia/` — blocos específicos do fluxo de denúncia
- `mapa/` — wrappers do Leaflet + a **alternativa textual** obrigatória do mapa
